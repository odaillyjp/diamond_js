# Diamond とは

サソリや毒ヘビなどが住む危険だらけのダイアモンド鉱山を探検し、これ以上進むのは危ないというぎりぎりのラインを見極めながらダイヤモンドを集め、一攫千金を目指すゲーム。
[http://sgrk.blog53.fc2.com/?no=144]

# Required

- Node.js

# How to play

```
$ node diamond.js
```
